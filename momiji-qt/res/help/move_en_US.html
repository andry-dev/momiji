MOVE

Synopsis

move.* SRC, DST

DST = SRC

Description

Copies data from SRC to DST.

Supported data types

Byte, Word and Long if DST is not an address register.

Word and Long if DST is an address register.

Equivalent C code

x = y;

Status register modifications

  • Overflow - 0
  • Carry - 0
  • Negative - Set if SRC is less than 0
  • Zero - Set if SRC is equal to 0
