MOVE

Sinossi

move.* SRC, DST

DST = SRC

Descrizione

Copia il contenuto di SRC in DST.

Tipi di dato supportati

Byte, Word e Long se DST non e' un registro indirizzi.

Word e Long se DST e' un registro indirizzi.

Codice C equivalente

x = y;

Modifiche al registro di stato

  • Overflow - 0
  • Carry - 0
  • Negative - Impostato se SRC e' minore di 0
  • Zero - Impostato se SRC e' uguale a 0
